# Cure Point Clinical Laboratory & Collection Point

A production-ready lab booking website for **Cure Point Clinical Laboratory & Collection Point**.

## Features
- **Bilingual Support**: English and Urdu (RTL) for all tests.
- **Real-time Search**: Filter tests instantly by name or Urdu translation.
- **Category Filters**: Easily browse tests by category (Infectious, Blood, Organ, etc.).
- **Multi-select Booking**: Select multiple tests and see them in a sidebar.
- **WhatsApp Integration**: Automatically generates a formatted booking message and opens WhatsApp.
- **Mobile First**: Fully responsive design optimized for mobile devices.
- **Modern UI**: Built with Tailwind CSS for a premium look and feel.
